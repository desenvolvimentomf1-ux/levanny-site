# Levany Rogge — Site Oficial

Site profissional de Levany Rogge, pedagoga e especialista em Proteção Infantil e Justiça Restaurativa, criadora do Método REAP+.

## Sobre o Projeto

Site estático em HTML5/CSS3/JavaScript vanilla, sem dependências de build. Pronto para hospedar diretamente na Vercel ou Netlify.

└── vercel.json                 ← Configuração de deploy e headers de segurança

## Como Hospedar na Vercel

1. Faça fork ou clone deste repositório
2. Acesse [vercel.com](https://vercel.com) e conecte com sua conta GitHub
3. Clique em "New Project" e selecione este repositório
4. Mantenha as configurações padrão (Framework: Other, Root Directory: /)
5. Clique em "Deploy"

O site vai ao ar em menos de 1 minuto.

## Como Hospedar no Netlify (alternativa)

1. Acesse [netlify.com](https://netlify.com)
2. Arraste a pasta do projeto para a área de deploy
3. Pronto — URL disponível instantaneamente
